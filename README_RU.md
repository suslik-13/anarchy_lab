# ANARCHY lab - Minetest/Luanti Server

## Описание

**ANARCHY lab** — это хардкорный PvP анархия-сервер для Minetest (Luanti) без правил и ограничений. Случайный спавн, система кланов и полная свобода действий!

### Особенности сервера:
- 🔥 **Режим анархии** — никаких правил, делай что хочешь
- ⚔️ **Хардкорный PvP** — сражайся с другими игроками
- 🎲 **Случайный спавн** — каждый раз новая локация
- 👥 **Система кланов** — создавай альянсы или воюй со всеми
- 🌍 **Открытый мир** — исследуй, строй, разрушай

**Discord сообщество:** https://discord.gg/JUFdNDWAcu

## Запуск сервера

### Требования
- Docker
- Docker Compose

### Команды запуска

Для запуска сервера используйте:

```bash
sudo docker compose up
```

Для запуска в фоновом режиме:

```bash
sudo docker compose up -d
```

Для остановки сервера:

```bash
sudo docker compose down
```

### Решение проблем с правами доступа

Если при запуске возникают ошибки связанные с правами доступа к файлам конфигурации или логам, например:

```
ERROR[Main]: Failed to open "/home/minetest/.minetest/debug.txt": Permission denied
ERROR[Main]: Could not read configuration from "/home/minetest/.minetest/minetest.conf"
```

Выполните следующую команду для исправления прав доступа:

```bash
sudo chmod -R 777 ./luanti_files
```

Эта команда устанавливает полные права доступа (чтение/запись/исполнение) на все файлы и директории внутри `luanti_files`, что позволяет контейнеру Docker свободно работать с конфигурацией, мирами и модами.

## Структура проекта

```
.
├── docker-compose.yml          # Конфигурация Docker Compose
├── luanti_files/               # Данные сервера (монтируется в контейнер)
│   ├── minetest.conf          # Главный конфиг сервера
│   ├── games/                 # Игровые моды и ресурсы
│   │   └── minetest_game/    # Основная игра с модами
│   ├── mod_data/              # Данные модов
│   └── worlds/                # Сохранённые миры
│       └── anarchy_lab/      # Мир сервера ANARCHY lab

```

## Сетевые порты

Сервер использует следующие порты:
- **30013/udp** — основной игровой порт
- **30013/tcp** — дополнительный TCP порт

Убедитесь, что эти порты открыты в вашем файерволе, если хотите, чтобы игроки могли подключаться извне.

## Конфигурация

Основной файл конфигурации находится в `./luanti_files/minetest.conf`. Вы можете редактировать настройки сервера, изменяя этот файл.

После изменения конфигурации перезапустите сервер:

```bash
sudo docker compose restart
```

## Моды

Сервер использует множество модов, включая:
- **3d_armor** — броня и защита
- **mobs_monster, mobs_animal** — мобы и существа
- **farming, cannabis** — сельское хозяйство
- **technic, hopper** — технические моды
- **serverfactions** — система кланов
- **stamina** — система выносливости
- **awards** — достижения
- И многие другие...

Полный список модов можно найти в директории `./luanti_files/games/minetest_game/mods/`.

### Discord relay (discordmt)

⚠️ **Важно:** Из сборки удален мод **discordmt**, так как он был модифицирован и содержал баги.

Если вам необходим Discord relay для связи игрового чата с Discord каналом:

1. Скачайте официальную версию мода **discordmt** с ContentDB: https://content.luanti.org/packages/archfan7411/discordmt/
2. Распакуйте мод в директорию `./luanti_files/games/minetest_game/mods/discordmt/`
3. Настройте мод согласно его документации
4. В файле `docker-compose.yml` раскомментируйте секцию `discordmt` (контейнер для Discord бота):
   ```yaml
   discordmt:
     image: python:3.11-slim
     container_name: discordmt
     # ... остальная конфигурация
   ```
5. Раскомментируйте также строку `depends_on: - discordmt` в секции `luanti`
6. Перезапустите сервер: `sudo docker compose up -d`

Официальная версия мода стабильна и не содержит известных багов модифицированной версии.
